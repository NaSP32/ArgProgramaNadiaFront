Portfolio Web - Argentina Programa

Mi primer proyecto como desarrolladora es el presente portfolio web. El mismo lo realicé en el marco del proyecto "Argentina Programa", que tuvo por objetivo desarrollar una aplicacion web que contuviese datos personales, estudios realizados, experiencia laboral, un login, y que permita con un usuario y contraseña, poder realizar la edicion de diferentes componentes como: "Persona, Educación, Empleos o experiencia laboral".
